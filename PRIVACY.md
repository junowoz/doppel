# Privacy Policy

Doppel is local-first software.

- No account is required.
- No internet connection is required for scanning or cleanup.
- No analytics are collected.
- No telemetry is collected.
- No tracking is performed.
- No file paths, hashes, metadata, or reports are uploaded.

The only in-app network feature is **Check for Updates**, which contacts official GitHub Releases after the user clicks it. Update checks do not upload file paths, hashes, metadata, or reports.

Reports exported by the user may contain file paths and hashes. Store and share exported reports carefully.
